Section 4 — MQTT Topics & RPC Methods
Dir Topic / Method Purpose
SUB v1/devices/me/rpc/request/+ Receive setLight / setRow / setAll commands
PUB v1/devices/me/rpc/response/{id} ACK command back to ThingsBoard
PUB v1/devices/me/telemetry state · kwh_used · rssi · uptime_s
PUB v1/devices/me/attributes lightState · ip · fw_ver on boot
RPC setLight / setRow / setAll Toggle 1 light · 1 row · all 36 lights
🛡️
FAIL-SAFE GUARANTEE — If WiFi disconnects, MQTT times out, or ESP32 reboots — the JQC3F relay automatically de-energizes. The Normally Closed (NC) contact closes, restoring AC power to the light. Light always defaults to ON when communication is lost — facility is never left in darkness.
